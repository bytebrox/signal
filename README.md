# SIGNAL

Smart money intelligence platform for Solana.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/signal)

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Solana Web3.js
